# Ducks Guts NS Tensor Engine

A high-performance Navier-Stokes solver with adaptive refinement, LES turbulence modeling, and a neural stability network. This code tackles turbulence at Re=10,000, achieving divergence below 1.10e-05 on grids up to 256^3.

## Features
- Solves 3D Navier-Stokes equations with LES (Smagorinsky model).
- Adaptive refinement from 64^3 to 256^3.
- Neural network (`StabilityNet`) for dynamic time step adjustment.
- Tested on xAI HyperCluster (64 NVIDIA H100 GPUs).

## Requirements
- Python 3.8+
- PyTorch 2.1 with CUDA 12.2
- NumPy, Matplotlib
- GPU cluster (minimum 16 GPUs recommended for 256^3 grids)
